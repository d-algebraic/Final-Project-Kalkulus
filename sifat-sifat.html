Sifat-Sifat Bilangan Kompleks

Himpunan bilangan kompleks (ℂ) bersama operasi penjumlahan dan perkalian membentuk bidang (field) kompleks. Sifat-sifat yang berlaku pada bilangan kompleks Z₁, Z₂, dan Z₃ adalah:

  1. Sifat Tertutup:

    Z₁ + Z₂ ∈ ℂ dan Z₁ ⋅ Z₂ ∈ ℂ.

  2. Sifat Komutatif:
    • Z₁ + Z₂ = Z₂ + Z₁
    • Z₁ ⋅ Z₂ = Z₂ ⋅ Z₁
  3. Sifat Asosiatif:
    • (Z₁ + Z₂) + Z₃ = Z₁ + (Z₂ + Z₃)
    • (Z₁ ⋅ Z₂) ⋅ Z₃ = Z₁ ⋅ (Z₂ ⋅ Z₃)
  4. Sifat Distributif:

    Z₁ ⋅ (Z₂ + Z₃) = (Z₁ ⋅ Z₂) + (Z₁ ⋅ Z₃).

  5. Elemen Netral Penjumlahan:

    Ada 0 = 0 + i0 ∈ ℂ, sehingga Z + 0 = Z.

  6. Elemen Netral Perkalian:

    Ada 1 = 1 + i0 ∈ ℂ, sehingga Z ⋅ 1 = Z.

  7. Elemen Invers Penjumlahan:

    Untuk setiap Z = x + iy ∈ ℂ, ada -Z = -x - iy, sehingga Z + (-Z) = 0.

  8. Elemen Invers Perkalian:

    Untuk setiap Z = x + iy ∈ ℂ, ada Z⁻¹ = 1 / Z, sehingga Z ⋅ Z⁻¹ = 1.
