Pengertian Bilangan Kompleks

Definisi 1

Bilangan kompleks adalah bilangan yang berbentuk: a + bi atau a + ib, di mana a dan b adalah bilangan real, dan i² = –1.

Bilangan kompleks dinyatakan dengan huruf z, sedangkan huruf x dan y menyatakan bilangan real. Jika z = x + iy menyatakan sembarang bilangan kompleks, maka:

  • x dinamakan bagian real dari z.
  • y dinamakan bagian imajiner dari z.

Bagian real dan bagian imajiner dari bilangan kompleks z biasanya dinyatakan dengan Re(z) dan Im(z).

Definisi 2

Bilangan kompleks z₁ = x₁ + iy₁ dan z₂ = x₂ + iy₂ dikatakan sama, z₁ = z₂, jika dan hanya jika:

  • x₁ = x₂.
  • y₁ = y₂.

Definisi 3

Untuk bilangan kompleks z₁ = x₁ + iy₁ dan z₂ = x₂ + iy₂, jumlah dan hasil kali mereka didefinisikan sebagai berikut:

  • Jumlah: z₁ + z₂ = (x₁ + x₂) + i(y₁ + y₂).
  • Hasil Kali: z₁ • z₂ = (x₁x₂ – y₁y₂) + i(x₁y₂ + x₂y₁).

Himpunan Bilangan Kompleks

Himpunan semua bilangan kompleks diberi notasi ℂ, sehingga: ℂ = { z | z = x + iy, x ∈ ℝ, y ∈ ℝ }.

Jika Im(z) = 0, maka bilangan kompleks z menjadi bilangan real x, sehingga bilangan real adalah keadaan khusus dari bilangan kompleks (ℝ ⊂ ℂ).

Jika Re(z) = 0 dan Im(z) ≠ 0, maka z menjadi iy dan dinamakan bilangan imajiner murni. Bilangan imajiner murni dengan y = 0, yaitu bilangan i, dinamakan satuan imajiner.
